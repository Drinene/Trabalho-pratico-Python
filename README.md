# Trabalho Prático - Persistência de Dados com Python

## Descrição

Este projeto foi desenvolvido com o objetivo de aplicar conceitos relacionados à manipulação de dados em Python, utilizando leitura e escrita de arquivos, estruturas de listas e algoritmos de ordenação.

Durante a atividade foram implementadas técnicas para leitura de arquivos no formato `.txt`, separação do conteúdo em palavras por meio do método `split()`, armazenamento dos dados em listas e aplicação de diferentes métodos de ordenação, como Bubble Sort, Selection Sort e o método nativo `sort()` do Python.

Além da implementação dos algoritmos, foi realizada uma análise comparativa do tempo de execução entre os métodos, permitindo identificar a solução com melhor desempenho para o processamento dos dados.

## Tecnologias utilizadas

- VS Code, py
- Git
- GitHub

## Arquivos do projeto

- `array.sort.py` → Ordenação de arrays utilizando `sort()`
- `bubble.sort.py` → Implementação do algoritmo Bubble Sort
- `selection.sort.py` → Implementação do algoritmo Selection Sort
- `ler.txt.py` → Leitura de arquivos `.txt`
- `escrever.txt.py` → Escrita de dados em arquivos `.txt`
- `kdd.py` → Forma prática dos conhecimentos apllicados nas outras atividades
- `loremipsum.txt` → Arquivo utilizado para leitura dos dados
- `palavras_ordenadas.txt` → Arquivo gerado com as palavras ordenadas
